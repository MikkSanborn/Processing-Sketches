Before running the program,
* Make sure you have all of the listed requirements


Requirements: Your Processing Environment must have the Sound library properly imported. To import the sound library, click on Sketch > Import Library > Add Library > Search "Sound" or "Provides a simple way to work with audio." > Select "Sound" > Click "Install"



To start showing the poem, click to start, once you see the album cover of Shine on, You Crazy Diamond.

Note: Project startup is fairly slow.
